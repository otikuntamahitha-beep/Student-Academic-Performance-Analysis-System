
# Student Academic Performance Analysis System

## Overview
A MySQL-based database management system designed to store, manage, and analyze student academic performance data.

## Features
- Student information management
- Teacher information management
- Enrollment tracking
- Academic record management
- Performance analysis using SQL queries
- Relational database design with ER Diagram

## Technologies Used
- MySQL
- SQL
- MySQL Workbench

## Project Files
- database_schema.sql
- sample_data.sql
- ER_Diagram.png

## Database Design
The ER Diagram illustrates the relationships between students, teachers, enrollments, academic records, and performance data.

## Learning Outcomes
- Database schema design
- SQL query writing
- Table relationships and normalization
- Data analysis using SQL
